# travelnest

Basic listing scraper for Airbnb.co.uk :

`$ python travelnest.py [-v]`

Built with python3 and pipenv but should be compatible with python2.7 and pip/virtualenv

Pipenv:

```
$ pipenv install
$ pipenv shell
(...) $ python travelnest.py [-v] 
(...) $ pytest tests.py [-v]
```

pip:

```
$ python setup.py develop
$ python travelnest.py [-v] 
$ pytest tests.py
```
